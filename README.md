# IEEE 控制算法研究枢纽 (IEEE Control Algorithm Research Hub)

## 项目愿景
本项目致力于建立一个开源的、高度协同的自动控制算法研究与开发平台，旨在为基于IEEE标准的自动控制算法研究提供一站式的中央调度和计算环境。
我们利用多个人工智能代理协作（包括 Hermes, Claude, Jules 等），实现控制算法模型的设计、仿真、优化以及验证环节的自动化和智能化，加速前沿控制理论与工业应用的结合。

## 架构概览
本系统采用模块化的中心辐射式架构（Hub and Spoke）：
- **中央调度模块 (Hub)**：通过 `hub.py` 实现对不同研究任务的调度、资源分配与执行跟踪。
- **AI代理协同层**：各具特长的AI协同完成代码生成、问题诊断、算法实现等任务。详细定义请见 [AGENTS.md](AGENTS.md)。
- **CI/CD 流水线**：借助 GitHub Actions 进行自动化测试和集成，确保代码质量和研究流程的连续性。
