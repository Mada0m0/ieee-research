# Stepping PZT motor leg voltage waveform design

## 1. Type and topology of motor legs

Stepper PZT motors are divided into three categories based on how the legs are driven:

### 1. Clamp-Drive Alternating
**Representative structure:** Bi-pedal driving alternately, each left and right leg contains a piezoelectric stack
- Left leg clamping → right leg driven propulsion → right leg clamping → left leg driven propulsion (four-phase gait)
- Paper: *Design and Experiment of a Clamping-Drive Alternating Operation Piezoelectric Actuator* (Micromachines, 2023, DOI: 10.3390/mi14030525)
- Features: Double flexible hinges to prevent overturning, double-layer stator structure, forward movement without regression

### 2. Inchworm
**Representative structure:** Clamping at both ends + middle drive
- Left clamping-right release → Drive to extend → Right clamping-Left release → Drive to retract
- Paper: *A Novel Piezoelectric Inchworm Actuator Driven by One Channel DC Signal* (IEEE TIE, 2020, DOI: 10.1109/tie.2020.2975493)
- Features: Single-channel DC signal simplified driving, stable step (0.241μm/step at 75V/3.2Hz)

### 3. Walker-Pusher
**Representative structure:** Two piezoelectric stacks are responsible for leg lifting and propulsion respectively
- Paper: *A walker-pusher inchworm actuator driven by two piezoelectric stacks* (MSSP, 2021, DOI: 10.1016/j.ymssp.2021.108636)
-Features: leg lift+propulsion decoupling, wide adjustable range of step length

### Detailed design of various waveforms

#### A. Sawtooth — stick-slip drive core

The most classic waveform for stick-slip piezo motor legs.

**Three elements of parametric design:**
1. **Rising edge slope** (slow expansion): determines the slow expansion/fast expansion ratio. Typical 4:1 ~ 10:1
   - The rising edge (t_rise) accounts for 80-90% of the cycle, and static friction drives the leg forward
   - The falling edge (t_fall) accounts for 10-20%, and the dynamic friction leg slides back
2. **Amplitude V_pp**: Determines the step size. Typical 13V (microstep) ~ 93V (high speed)
   - Actual measurement in the paper: 3.5μrad/step @ 13V/3000Hz | 0.44rad/s @ 93V/3000Hz
3. **Frequency f**: affects the consistency of speed and step distance
   - Low frequency (<100Hz): large step size but obvious jitter
   - Intermediate frequency (1-5kHz): the best balance area
   - High frequency (>10kHz): step attenuation, approaching continuous motion

                        ↑       ↑       ↑       ↑

**Key design parameters:**
- **Phase Overlap** (overlap): Adjacent phases overlap by 5-15% to prevent simultaneous release and sliding back
- **Rising/falling slope**: Typical 50-100V/ms, too steep will cause impact, too slow will reduce the step size
- **Hold voltage V_hold**: Clamp leg holding voltage (typically 30-50% V_max), reducing creep
- **Adjustable duty cycle**: Symmetrical mode (25% each) | Asymmetrical (propulsion phase is longer than clamping phase)

#### D. DC Single Channel Driver – Latest Trend

The innovative solution proposed by *IEEE TIE 2020*: Use only one DC signal to drive the inchworm motor.
- Clamping force generated by DC motor + permanent magnet (non-piezoelectric)
- The piezoelectric stack is triggered by a photoelectric sensor detecting the position of the magnet
- Advantages: minimalist drive circuit, no rollback, stable step size 0.241μm
- Limitations: speed limited (up to 3.2Hz), mechanical sensor required

#### E. Sine + Bias — Ultrasonic Motor Mode

Ultrasonic motors use two-phase or multi-phase sine wave drive to produce continuous motion through standing/traveling waves:
```
Phase A: V_a = V_0 * sin(ωt)
Phase B: V_b = V_0 * sin(ωt + φ)

φ = 90° → traveling wave (rotation)
φ = 0° → standing wave (vibration)
Offset V_DC Adjustable Zero
```

## 3. Key design formulas

### Step distance estimation

**Clamp-driven type:**
```
Δx_step = n * d33 * V_drive
in:
  n = number of piezoelectric stack layers
  d33 = Piezoelectric coefficient (pm/V)
  V_drive = drive voltage (V)
```

**Sticky type:**
```
Δx_step = (d33 * V_pp) * (1 - t_fall / t_rise)
         ≈ d33 * V_pp * k
in:
  k = 1 - (t_fall/t_rise), typical value 0.6-0.9
  t_rise = (0.8-0.9) * T
  t_fall = (0.1-0.2) * T
```

### Speed ​​estimate

```
v = Δx_step * f * η
in:
  f = drive frequency (Hz)
  η = efficiency coefficient (0.5-0.8, depending on load and friction)
```

### Clamping force design

```
F_clamp = k_preload * Δx_preload + F_electrostatic
in:
  k_preload = preload spring stiffness
  Δx_preload = pre-compression amount
  F_electrostatic ≈ ε * A * (V_clamp / d)^2 / 2
```

## 5. Recommended waveform selection matrix

| Motor Type | Recommended Waveforms | Reasons |
|---------|---------|------|
| Bi-foot clamping and driving | Trapezoidal wave (four phases) | Phase controllable, no fallback |
| Inchworm type | Trapezoidal wave (three phases) | Clamp → drive → release sequence |
| Stick-slip type | Sawtooth wave | Principle of slow expansion and quick contraction |
| Walking-propulsion type | DC+ modulation | Leg lift/propulsion decoupling |
| Ultrasonic type | Sine wave (polyphase) | Resonance drive |

### Actual waveform tuning suggestions

1. **Prioritize optimization of rising/falling slope** → Determine step consistency
2. **Phase Overlap 5-15%** → Eliminate Backward Movement
3. **Sweep the voltage amplitude from small to large** → Find the resonance point (if any)
4. **Microstepping mode test N=8/16** → smoothness vs speed trade-off
5. **Frequency scan 1Hz→10kHz** → Determine the best working range
6. **Temperature compensation V_T** → Adjust V_pp 3-5% every 10°C
